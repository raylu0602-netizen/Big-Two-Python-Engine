🃏 Big Two Python Engine (大老二遊戲引擎)這是一個基於 Python 開發的大老二 (Big Two) 遊戲引擎，具備完整的牌型判定邏輯、自動化回合管理，以及基於啟發式搜尋的 AI 對手。
🚀 核心技術亮點1. 公平洗牌演算法 (O(n))
    
    1.採用 Fisher-Yates Shuffle 演算法，確保 52 張牌的每一種排列組合出現機率均等。
        時間複雜度：O(n)，僅需一次遍歷即可達成完美隨機。
        數學證明：每張牌出現在每個位置的機率皆為 1/n。
    2. 物件導向設計 (OOP)
        專案架構清晰，將遊戲組件高度抽象化：Card: 定義權重系統（如 2 最大的特殊邏輯）。
        Hand: 封裝牌型判定與強度比較。
        Player: 支援 AI 搜尋邏輯與真人互動模式。
    3.進階牌型判定：循環順子演算法
        針對大老二特殊的順子規則（如 A-2-3-4-5），開發了基於**平移（Shift）與集合（Set）**的校驗邏輯：唯一性檢查：利用 set() 確保 5 張點數不重複，杜絕邏輯漏洞。
        循環位階處理：處理 A, 2 與 3 之間的斷層，確保判定的嚴謹性。
    4.啟發式 AI 決策
        AI 具備基礎的戰略思維：特定牌型優先搜尋：優化搜尋空間，從 $O(\binom{n}{5})$ 降至特定組件匹配。
        保留實力策略：具備大牌保留邏輯，防止在遊戲前期過早交出關鍵牌（如 2）。
